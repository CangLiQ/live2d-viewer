# Live2D Viewer

一个基于 Vue 3 + Vite 的 Live2D 模型查看器，支持在浏览器中加载和浏览本地的 Live2D 模型文件。

## 功能

***主要用于快速浏览Mygo假药WebGal二创的本地Live2D模型，目前仅测试。***

- 浏览本地 Live2D 模型文件夹
- 支持 Cubism 2.1 模型
- 模型动作播放与控制
- 表情切换
- 分类标签管理

## 在线体验

https://cangliq.github.io/live2d-viewer/

## 技术栈

- Vue 3 + TypeScript
- Vite
- Pinia（状态管理）
- Pixi.js + pixi-live2d-display
- Naive UI
- Tailwind CSS
